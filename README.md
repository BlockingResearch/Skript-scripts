# Skript Scripts

Colección de scripts de Skript para Minecraft que puedes usar en tus eventos.
## Instalación

### Requisitos
- **Minecraft** 1.13+
- **Skript** 2.6+ ([Descargar](https://github.com/SkriptLang/Skript/releases))
- + Addons (Depende del script)

### Pasos
1. Descarga e instala el plugin [Skript](https://github.com/SkriptLang/Skript/releases) y los addons necesarios
2. Coloca los archivos `.sk` en la carpeta `plugins/Skript/scripts/`
3. Recarga los scripts con `/skript reload`

## Solución de Problemas

### Los scripts no cargan
- Verifica que Skript esté instalado correctamente
- Ejecuta `/skript reload` para recargar
- Revisa la consola para errores
- Skbee: Algunas versiones desactualizadas de este addon puede corromper funcionalidades
